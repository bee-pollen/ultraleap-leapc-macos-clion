# Ultraleap LeapC on macOS (Apple Silicon) – CLion Example

This repository demonstrates how to build and run **Ultraleap LeapC (C API)** on **macOS Apple Silicon (M1/M2/M3)** using **CLion and CMake**.

✅ No Python  
✅ Native C  
✅ Works with Ultraleap Hand Tracking (Gemini / Hyperion)

---

## Requirements

### Hardware
- Apple Silicon Mac (M1/M2/M3)
- Leap Motion Controller (Controller 2 recommended)

### Software
- Ultraleap Hand Tracking App
- CLion
- Xcode Command Line Tools
